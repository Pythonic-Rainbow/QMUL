About me

Got interested in computers because my first laptop couldn't handle games well. I started researching factors impacting a system's performance and ultimately built my own computer at 17.

I taught myself Python because I wanted to make a bot on Discord. However, with no prior coding experience, the code quickly became a mess. That didn't stop me because I was fascinated at how programs would behave when we write code, and so I would search on Google hundreds of time everyday. I've had rewritten the bot multiple times, but with each rewrite I felt like I've learnt a lot. I eventually learnt multiple programming languages with the help of Google and many other online resources.

Personal projects & skills

DimBot : A Discord bot, highly concurrent. Base64 encode/decode, automatic RSS/YouTube feeds relay, auto chat moderation, stream music from YouTube, remotely starts a DigitalOcean cloud instance and more.
Runs on a self-managed AWS EC2 cloud instance.

Lokeon : A plugin written in Java for Minecraft Spigot servers. Sends HTTP requests to DimBot when certain in-game events are fired. Also contains a timer that shuts down the Linux server its running on if there are no players in the server for a defined duration.
Runs on a self-managed DigitalOcean Droplet cloud instance.

BigBangMods-Discord : Gradle-based mod for Minecraft Forge/Fabric written in Java. Allows players to display their currently playing game as my modpack. Uses jar shadowing to shade dependencies into one fat jar.
Available on CurseForge.

RbxTBRnd : A simple website that randomises towers and maps for a tower defense game.
Hosted on GitHub Pages.

Luigi Poker : An attempt to recreate the old 'Luigi Poker' minigame found in 2 Mario games for the Nintendo DS. Uses Python aiohttp as a HTTPS+WebSocket server. Contains a SSL certificate authorised by Let's Encrypt.
Runs on DimBot's AWS EC2 cloud instance with a custom subdomain via afraid.org

PyCurseForge : A Python wrapper for the unofficial CurseForge endpoint.
